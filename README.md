# 🍊 No-Code Data Mining & Analytics using Orange

Repositori ini berisi dokumentasi proyek analisis data menggunakan **Orange Data Mining**, sebuah platform berbasis visual programming (*no-code/low-code*) standar industri. Proyek ini berfokus pada pemodelan alur kerja data (*data workflow*) untuk melakukan klasifikasi, klasterisasi, dan eksplorasi visual pada data penjualan.

---

## 🛠️ Alat & Metodologi
* **Orange Data Mining Software:** Platform pemrograman visual berbasis *widgets* untuk pemrosesan data, visualisasi, dan pemodelan machine learning.
* **Workflow Komponen:** Memanfaatkan widget bawaan seperti *File Ingestion, Data Table, Scatter Plot, Distributions,* hingga evaluasi model.

---

## 📈 Arsitektur Alur Kerja (Data Workflow)
Proyek ini mengimplementasikan alur pemrosesan data visual (*visual programming pipeline*) yang terstruktur:

1. **Data Loading & Inspection:** Mengintegrasikan dataset eksternal ke dalam sistem Orange dan memeriksa struktur tabel serta tipe variabel (fitur numerik vs kategorikal).
2. **Eksplorasi Visual (Data Visualization):** Menggunakan widget *Distributions* untuk melihat persebaran jumlah barang terjual dan widget *Scatter Plot* untuk menganalisis korelasi antar variabel bisnis secara visual.
3. **Analisis Segmentasi Pasar:** Melakukan pemetaan karakteristik data penjualan untuk melihat pola perilaku transaksi atau pengelompokkan jenis produk berdasarkan performa rating dan harga.
